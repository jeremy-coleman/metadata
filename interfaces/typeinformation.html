params

params: TypeInformation[]
  • Defined in index.ts:43

Type parameters if this type is generic. For example, string[] will be represented by { type: Array, params: [{ type: String }] }. This can be an empty array if there is no type parameter.

type

type: Type
  • Defined in index.ts:29

Class constructor for the main type. A string type is represented by the global StringConstructor, and an array is represented by the global ArrayConstructor, etc.

This is a getter property and will trigger a ReferenceError if the class is in temporal dead zone.

typeFactory

typeFactory: () => Type
  • Defined in index.ts:36

Function that returns the class constructor. Most libraries that use decorator metadata accept a type factory function, which is provided here.
